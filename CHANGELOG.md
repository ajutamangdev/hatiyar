## [1.0.0] - 2025-11-09

- CVE exploit modules (CVE-2021-42013, CVE-2021-43798)
- Interactive CLI shell with tab completion
- Rich terminal UI with colors and formatting
- Module registry system with YAML configuration
- Improved command reference table with Rich formatting
- Enhanced UX with visual indicators
